# AWS AI Marketing Platform

An AI-powered e-commerce and personalized marketing platform with admin control.

## Features
- User authentication (Login / Signup)
- Product browsing with AI recommendations
- Cart & checkout system
- Admin product management (Add / Edit / Delete)
- AI campaign generation & deployment
- Campaign history dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Cloud: AWS EC2, DynamoDB, SNS (Simulated)
- Version Control: Git & GitHub

## How to Run Locally
```bash
pip install -r requirements.txt
python app.py
